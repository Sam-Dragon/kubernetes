# Limit Range
- It allows us to configure
  1. Default memory and cpu
  2. Minimum memory and cpu
  3. Maximum memory and cpu

apiVersion: v1
kind: LimitRange
metadata:
  name: default-max-minlimit-range
spec:
  limits:
  - default:
      memory: 512Mi
    defaultRequest:
      memory: 256Mi
    default:
      cpu: 800m
    defaultRequest:
      cpu: 800m
    max:
      memory: 1Gi
    min:
      memory: 500Mi
    max:
      cpu: "800m"
    min:
      cpu: "200m" 
    type: Container
